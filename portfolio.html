GPA Caculator

In my program, abstraction is used in many ways. From the users perspective, all the user

needs to know is how to initiate the program with enter() and how to answer the questions

using letters and numbers and the program will eventually spit out a result. To the user,

most of the code is abstraction to them because they don't care how it does it, they just

care that it does what it is supposed to do. From the programmers point of view, many

mathematical equations and other things are abstraction to the programmer, because

like the user to the code, the programmer only cares that the code does what the programmer

told it to do, and not what the code tells the software to do and so forth.
